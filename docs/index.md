# MadCAD Setup

Repositório para configuração e troubleshooting do pymadcad/uimadcad.

## Quick Start

```bash
# Install
pip install pymadcad==0.19.1
pip install uimadcad
pip install processional

# Run
python -m uimadcad
```

## Problemas Conhecidos

- [ ] brick() NaN bug
- [ ] GLSL 4.30 não suportado
- [ ] note_* funções não exportadas

- `run_example_cli.sh` - Rodar exemplos via CLI
- `kill_madcad.sh` - Matar processos
