# Scripts

## run_example_cli.sh

Roda exemplos via CLI com QApplication.

```bash
./run_example_cli.sh <exemplo.py> [tema]

# Exemplos
./run_example_cli.sh pymadcad/examples/bearing.py
./run_example_cli.sh pymadcad/examples/elliptic-gearbox.py dark-green
```

Temas: `grey-orange`, `dark-red`, `dark-green`
